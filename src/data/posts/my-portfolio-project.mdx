# Next.js ile Modern Portföy Sitesi Nasıl Yapılır?

Bu yazıda, kendi portföy sitemin nasıl yapıldığını ve kullanılan teknolojileri detaylı bir şekilde anlatacağım. Proje Next.js 14, TypeScript, Tailwind CSS ve shadcn/ui kullanılarak geliştirildi.

## Proje Yapısı

Projemiz aşağıdaki temel yapıya sahip:

```bash
src/
  ├── app/           # Next.js app router sayfaları
  ├── components/    # React bileşenleri
  ├── data/         # Blog yazıları ve statik veriler
  ├── lib/          # Yardımcı fonksiyonlar
  └── types/        # TypeScript tip tanımlamaları
```

## Kullanılan Teknolojiler

### 1. Next.js 14 App Router

Next.js'in yeni App Router özelliğini kullanarak, sayfa yönlendirmelerini ve layout yapısını oluşturduk. İşte ana layout dosyamızdan bir örnek:

```tsx
// src/app/layout.tsx
import { Navbar } from '@/components/shared/navbar';

export default function RootLayout({ children }: { children: React.ReactNode }) {
  return (
    <html lang="en" suppressHydrationWarning>
      <body>
        <Navbar />
        <main>{children}</main>
      </body>
    </html>
  );
}
```

### 2. Özel UI Bileşenleri

Projede shadcn/ui kütüphanesini temel alarak özel bileşenler geliştirdik. İşte navbar bileşeninden bir örnek:

```tsx
// Navbar bileşeni örneği
export function Navbar() {
  const [isScrolled, setIsScrolled] = useState(false);

  useEffect(() => {
    const handleScroll = () => {
      setIsScrolled(window.scrollY > 10);
    };

    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, []);

  return (
    <div className="pointer-events-none fixed inset-x-0 top-0 z-30">{/* Navbar içeriği */}</div>
  );
}
```

### 3. Blog Sistemi

Blog yazıları MDX formatında yazılıyor ve Next.js'in yerleşik MDX desteği ile işleniyor:

```tsx
// src/lib/mdx.ts
import { compileMDX } from 'next-mdx-remote/rsc';
import rehypeAutolinkHeadings from 'rehype-autolink-headings';
import rehypeHighlight from 'rehype-highlight';
import rehypeSlug from 'rehype-slug';

export async function getMDXContent(source: string) {
  const { content, frontmatter } = await compileMDX({
    source,
    options: {
      parseFrontmatter: true,
      mdxOptions: {
        rehypePlugins: [
          rehypeSlug,
          rehypeHighlight,
          [rehypeAutolinkHeadings, { behavior: 'wrap' }],
        ],
      },
    },
  });

  return { content, frontmatter };
}
```

### 4. Animasyonlar ve Efektler

Projede çeşitli animasyonlar ve efektler için Framer Motion ve özel CSS kullanıldı:

```tsx
// src/components/magicui/blur-fade.tsx
'use client';

import { motion } from 'framer-motion';

export function BlurFade({ children }: { children: React.ReactNode }) {
  return (
    <motion.div
      initial={{ opacity: 0, filter: 'blur(10px)' }}
      animate={{ opacity: 1, filter: 'blur(0px)' }}
      transition={{ duration: 0.5 }}
    >
      {children}
    </motion.div>
  );
}
```

## Özelleştirme ve Tema Desteği

Proje, Tailwind CSS ile tam özelleştirilebilir bir tema sistemine sahip:

```ts
// tailwind.config.ts
import type { Config } from 'tailwindcss';

const config: Config = {
  darkMode: ['class'],
  content: [
    './src/pages/**/*.{ts,tsx}',
    './src/components/**/*.{ts,tsx}',
    './src/app/**/*.{ts,tsx}',
  ],
  theme: {
    extend: {
      colors: {
        border: 'var(--border)',
        background: 'var(--background)',
        foreground: 'var(--foreground)',
      },
    },
  },
};

export default config;
```
