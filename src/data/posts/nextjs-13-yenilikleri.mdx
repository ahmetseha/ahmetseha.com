# Next.js 13 Yenilikleri

Next.js 13, React uygulamaları geliştirmek için kullanılan popüler framework'ün en yeni sürümü. Bu yazıda, Next.js 13 ile gelen önemli yenilikleri ve değişiklikleri inceleyeceğiz.

## App Directory

Next.js 13'ün en önemli yeniliklerinden biri `app` directory yapısı. Bu yeni yapı ile:

- Daha iyi dosya organizasyonu
- Layout'lar için daha iyi destek
- Server Components varsayılan olarak aktif
- Daha iyi routing deneyimi

```jsx
// app/layout.tsx
export default function RootLayout({ children }) {
  return (
    <html lang="tr">
      <body>{children}</body>
    </html>
  );
}
```

## Server Components

React Server Components artık Next.js 13'te varsayılan olarak aktif. Bu sayede:

- Daha hızlı sayfa yüklemeleri
- Daha az JavaScript bundle boyutu
- SEO dostu yapı
- Backend kaynaklarına doğrudan erişim

```jsx
// Bu bir server component
async function PostList() {
  const posts = await getPosts(); // Doğrudan veritabanına erişebilirsiniz

  return (
    <ul>
      {posts.map((post) => (
        <li key={post.id}>{post.title}</li>
      ))}
    </ul>
  );
}
```

## Turbopack

Next.js 13 ile birlikte gelen Turbopack, Webpack'e göre:

- 700x daha hızlı başlangıç
- 10x daha hızlı güncelleme
- Daha az bellek kullanımı

## Sonuç

Next.js 13, modern web uygulamaları geliştirmek için gereken birçok önemli özelliği beraberinde getiriyor. Server Components ve yeni `app` directory yapısı ile daha performanslı ve organize uygulamalar geliştirebilirsiniz.
