# TypeScript ile Tip Güvenli Programlama

TypeScript, JavaScript'in üzerine inşa edilmiş, statik tip tanımlamaları ekleyen bir programlama dilidir. Bu yazıda, TypeScript'in sağladığı tip güvenliği özelliklerini ve bunların kod kalitemizi nasıl artırdığını inceleyeceğiz.

## Tip Tanımlamaları

TypeScript'te değişkenler, fonksiyonlar ve sınıflar için tip tanımlamaları yapabiliriz:

```typescript
// Değişken tip tanımlaması
let isActive: boolean = true
let count: number = 42
let name: string = "Ahmet"

// Fonksiyon tip tanımlaması
function greet(name: string): string {
  return `Merhaba, ${name}!`
}

// Sınıf tip tanımlaması
class User {
  id: number
  name: string

  constructor(id: number, name: string) {
    this.id = id
    this.name = name
  }
}
```

## Interface ve Type

TypeScript'te karmaşık veri yapılarını tanımlamak için interface ve type kullanabiliriz:

```typescript
// Interface tanımlaması
interface User {
  id: number
  name: string
  email: string
  isActive?: boolean // Opsiyonel alan
}

// Type tanımlaması
type Status = "active" | "inactive" | "pending"
```

## Generic Types

TypeScript'in güçlü özelliklerinden biri, generic tiplerdir:

```typescript
function getFirstElement<T>(array: T[]): T | undefined {
  return array.length > 0 ? array[0] : undefined
}

// Kullanımı
const numbers = getFirstElement<number>([1, 2, 3]) // number | undefined
const strings = getFirstElement<string>(["a", "b", "c"]) // string | undefined
```

## Sonuç

TypeScript, büyük ölçekli uygulamalarda kod kalitesini artırmak ve hataları erken tespit etmek için mükemmel bir araçtır. Tip güvenliği sayesinde, daha güvenilir ve sürdürülebilir kod yazabiliriz.
